# 学业预警系统 - 数据库设计文档

> 本文档基于 Django Models 代码自动生成，覆盖 `back/backend/` 下全部数据模型。

---

## 一、ER 图文字描述

### 1.1 用户体系模块

```
auth_user (Django 内置用户认证表)
    ├── Teacher (1:1)  扩展教师业务信息
    │   └── teachers 表: teacher_no, department, title, phone, office
    ├── Counselor (1:1)  扩展辅导员业务信息
    │   └── counselors 表: employee_no, department, phone, office
    └── UserProfile (1:1)  扩展学生/未分类用户信息
        └── user_profiles 表: employee_no, department, phone
```

### 1.2 教学管理模块

```
Major (专业)
    └── Class (班级)  N:1
        ├── counselor_id → auth_user.id  (辅导员管理班级)
        └── Student (学生)  N:1
            ├── student_no (唯一学号)
            ├── class_id → classes.id
            └── major_id → majors.id

Teacher
    └── Course (课程)  N:1
        ├── teacher_id → teachers.user_id
        ├── semester
        └── KnowledgePoint (知识点)  N:1
            └── parent_id 自关联 (知识点层级)

Course
    └── CourseEnrollment (选课关系)  1:N
        ├── student_id → students.id
        ├── course_id → courses.id
        └── unique(student_id, course_id)
```

### 1.3 学习活动模块

```
Course
    ├── HomeworkAssignment (作业任务)  1:N
    │   └── HomeworkSubmission (作业提交)  1:N
    │       ├── assignment_id → homework_assignments.id
    │       ├── student_id → students.id
    │       └── unique(assignment, student)
    └── ExamAssignment (考试任务)  1:N
        └── ExamResult (考试结果)  1:N
            ├── exam_id → exam_assignments.id
            ├── student_id → students.id
            └── unique(exam, student)

Student + Course
    └── LearningActivity (学习活动)  N:1 + N:1
        ├── student_id → students.id
        ├── course_id → courses.id
        └── chapter_id → knowledge_points.id (可选)
```

### 1.4 预警分析模块

```
Student + Course
    └── StudentCourseScore (学生课程综合得分)  N:1 + N:1
        ├── unique(student, course)
        ├── attendance_rate, video_progress, homework_avg, exam_avg
        └── knowledge_mastery

StudentCourseScore
    └── WarningRecord (预警记录)  1:N
        ├── student_id → students.id
        ├── course_id → courses.id (可选, 综合预警时为空)
        ├── risk_level: high/medium/low/normal
        ├── composite_score (综合得分)
        └── status: active/resolved/ignored
```

### 1.5 干预管理模块

```
WarningRecord
    └── InterventionRecord (干预记录)  1:N
        ├── student_id → students.id
        ├── course_id → courses.id (可选)
        ├── intervenor_id → auth_user.id (干预执行人)
        ├── warning_id → warning_records.id (可选)
        └── intervention_type: talk/academic/psychological/family/other

WarningRecord
    └── SMSNotification (短信通知)  1:N
        ├── student_id → students.id
        ├── sender_id → auth_user.id
        ├── warning_id → warning_records.id (可选)
        └── status: pending/sent/delivered/failed
```

### 1.6 数据导入模块

```
ImportLog (导入记录)
    ├── import_type: activities/homework/exams/enrollments
    ├── course_id → courses.id (可选)
    ├── status: pending/success/partial/failed
    └── total_rows / success_rows / failed_rows
```
