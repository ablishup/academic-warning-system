#### 已创建API

**用户认证** (`/api/auth/`)
- POST `/login/` - 登录
- POST `/logout/` - 登出
- GET `/current/` - 当前用户
- GET `/list/` - 用户列表

**课程管理** (`/api/courses/`)
- GET `/` - 课程列表
- GET `/<id>/` - 课程详情
- GET `/<id>/knowledge-points/` - 知识点
- GET `/teacher/` - 教师课程
- GET `/student/?student_id=x` - 学生课程

**学生班级** (`/api/classes/`)
- GET `/students/` - 学生列表
- GET `/students/search/?q=keyword` - 搜索
- GET `/students/<id>/` - 学生详情
- GET `/classes/` - 班级列表
- GET `/classes/<id>/students/` - 班级学生
