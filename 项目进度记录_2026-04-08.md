#### 已创建API

**用户认证** (`/api/auth/`)
- POST `/login/` - 登录
- POST `/logout/` - 登出
- GET `/current/` - 当前用户
- GET `/list/` - 用户列表

**课程管理** (`/api/courses/`)
- GET `/` - 课程列表
- GET `/<id>/` - 课程详情
- GET `/<id>/knowledge-points/` - 知识点
- GET `/teacher/` - 教师课程
- GET `/student/?student_id=x` - 学生课程

**学生班级** (`/api/classes/`)
- GET `/students/` - 学生列表
- GET `/students/search/?q=keyword` - 搜索
- GET `/students/<id>/` - 学生详情
- GET `/classes/` - 班级列表
- GET `/classes/<id>/students/` - 班级学生

**预警系统** (`/api/warnings/`) ✅ (2026-04-08)
- GET `/` - 预警列表（支持筛选：risk_level, status, student_id, course_id）
- GET `/<id>/` - 预警详情
- POST `/calculate/` - 计算预警（加权评分：出勤30%+进度20%+作业30%+考试20%）
- POST `/<id>/resolve/` - 解决预警
- GET `/stats/` - 预警统计
- GET `/scores/` - 学生课程综合得分

## 算法决策变更

**预警算法**：XGBoost → **随机森林**
- 原因：随机森林更易于解释，适合毕业设计答辩展示
- 权重：出勤30%、进度20%、作业30%、考试20%
- 等级：<60红、60-75橙、75-85黄、≥85正常
