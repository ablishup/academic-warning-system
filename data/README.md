# 基于在线教学平台的数据挖掘与学习行为分析超星集团数据集

## 简介

本仓库提供了一个名为“基于在线教学平台的数据挖掘与学习行为分析超星集团数据集”的资源文件。该数据集旨在帮助研究人员和数据科学家分析在线教学平台上的学习行为，并通过数据挖掘技术揭示潜在的学习模式和趋势。

## 数据集描述

该数据集包含了超星集团在线教学平台上的大量学习行为数据。通过分析这些数据，研究人员可以深入了解学生的学习习惯、参与度、以及课程内容的有效性。数据集的具体内容包括但不限于：

- 学生登录记录
- 课程访问记录
- 作业提交情况
- 在线测试成绩
- 讨论区互动数据

## 使用场景

该数据集适用于以下研究方向：

- 在线教育平台的用户行为分析
- 学习行为模式挖掘
- 个性化学习推荐系统开发
- 教育数据挖掘与预测分析

## 注意事项

- 数据集仅供研究使用，请勿用于商业目的。
- 在使用数据集时，请遵守相关法律法规和伦理规范。
- 数据集可能包含敏感信息，请确保在处理和分析过程中保护用户隐私。

## 数据处理流程（用于学业预警系统）

### 快速开始

```bash
# 1. 安装依赖
pip install pandas pymysql openpyxl requests

# 2. 执行数据清洗
python scripts/clean_data.py

# 3. 创建数据库表
mysql -u root -p academic_warning_system < ../backend/sql/chaoxing_schema.sql

# 4. 导入数据
python scripts/import_to_mysql.py

# 5. 计算预警
python scripts/calculate_warnings.py
```
