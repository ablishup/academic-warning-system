## 快速开始

### 1. 启动后端服务

```bash
cd back/backend
python manage.py runserver
```

### 2. 启动前端服务

```bash
cd forward
npm run serve
```

### 3. 访问系统

打开浏览器访问 http://localhost:8080

### 4. 登录

使用上表中的任意账号和密码登录系统。

## 测试建议

### 辅导员端测试
- 使用 `counselor` 或 `counselor2` 登录
- 查看预警学生列表
- 选择一个高危学生生成AI评语
- 点击"添加干预记录"跳转到干预页面
- 验证弹窗中是否显示学生信息和AI评语参考

### 教师端测试
- 使用 `teacher1` 或 `teacher2` 登录
- 查看课程列表和学生
- 尝试上传成绩数据

### 学生端测试
- 使用 `teststudent` 或真实学号登录
- 查看个人学情分析
- 查看课程资源和AI评语

## 安全提示

- 所有测试账号密码均为简单密码，**仅用于开发和测试环境**
- 生产环境请使用强密码并定期更换
- 真实学生数据请妥善保管，遵守隐私保护法规
- 如需创建更多测试账号，可使用管理员账号在后台创建

## 数据重置

如需重置数据库到初始状态：

```bash
cd back/backend
python manage.py flush
python manage.py loaddata initial_data  # 如有fixture文件
```
